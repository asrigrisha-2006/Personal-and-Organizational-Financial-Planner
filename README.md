#  Personal and Organizational Financial Planner

A full-stack financial management web application designed for individuals and organizations to efficiently manage income, expenses, savings, investments, and financial goals through an interactive dashboard.

---

##  Features

-  Secure user authentication using bcrypt
-  Income and expense management
-  Savings and investment tracking
-  Financial goal management
-  Interactive dashboard with financial analytics
-  Personalized savings and investment recommendations
-  Financial reports for better budgeting

---

##  Tech Stack

**Backend**

##  Installation

Clone the repository

```bash
git clone https://github.com/asrigrisha-2006/Personal-and-Organizational-Financial-Planner.git
```

Install dependencies

```bash
npm install
```

Configure MySQL

Update your database credentials in:

```text
config/db.js
```

Run the application

```bash
node server.js
```
